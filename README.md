# bookbot
---

So you tired manually counting each letters and words in the book?
*Bookbot* will help you to your weird problems!
It can do:
- can count all words
- can count all characters

## (>w<) Quick Start

Python 3.8 or higher is required

### 1. Clone the repo

```bash
git clone https://github.com/tianchongfengsula/bookbot.git
```

### 2. Test it using the example txts (thanks gutenberg!)

```bash
python3 bookbot/main.py bookbot/books/prideandprejudice.txt
```
